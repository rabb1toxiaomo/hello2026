## 🎯 项目描述

一个精美的元旦/春节祝福留言系统，支持实时留言、弹幕、抽奖、排行榜等功能。采用液态玻璃和苹果风格的毛玻璃UI设计，包含多种炫酷动画效果。

## 🛠️ 技术栈

- **后端**: Node.js + Express + SQL.js
- **前端**: 原生 JavaScript + HTML5 + CSS3
- **数据库**: SQLite (SQL.js)
- **部署**: PM2 / Docker
